# 🏥 Healthcare Data Analysis Dashboard

## 📌 Project Description
This project focuses on analyzing healthcare data using Power BI to gain meaningful insights from patient records. The dashboard helps in understanding patient demographics, medical conditions, hospital performance, and overall service quality.

The goal of this project is to transform raw healthcare data into interactive visualizations that support better decision-making.

---

## 🎯 Objectives
- To analyze patient data efficiently
- To identify patterns in hospital admissions
- To understand patient distribution based on gender and medical conditions
- To evaluate hospital performance using key metrics
- To present insights in a clear and interactive dashboard

---

## 📊 Key Features / Insights
- 👥 **Total Patients Count**: Displays the total number of patients
- 🚻 **Gender Distribution**: Shows male vs female patient ratio
- 🏥 **Condition-wise Analysis**: Identifies most common medical conditions
- ⏳ **Average Length of Stay**: Shows how long patients stay in hospital
- 🔁 **Readmission Analysis**: Tracks how many patients are readmitted
- 😊 **Patient Satisfaction**: Displays satisfaction levels of patients

---

## 📁 Dataset Information
The dataset used in this project is in CSV format and includes the following columns:

- Patients ID
- Age
- Gender
- Condition
- Length of Stay
- Outcomes
- Procedure
- Readmission
- Satisfaction

---

## 🛠 Tools & Technologies Used
- Power BI (for data visualization)
- CSV (for dataset storage)

---

## 📈 Dashboard Overview
The Power BI dashboard includes multiple visualizations such as:

- Card (for KPIs like total patients)
- Donut Chart (for gender distribution)
- Bar Chart (for condition-wise analysis)
- Column Chart (for length of stay)
- Slicers (for filtering data interactively)

## 🚀 How to Use This Project
1. Download the dataset (CSV file)
2. Open Power BI Desktop
3. Import the dataset using "Get Data"
4. Create visualizations as shown in the dashboard
5. Use slicers to interact with the data
